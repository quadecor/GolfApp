# ⛳ Golf Players Manager

A web application for managing golf players and tracking their average scores, with real-time cloud database synchronization using Firebase.

## Features

- ✅ Add, edit, and delete players
- ✅ Track average scores for each player
- ✅ Automatic ranking system (best score first)
- ✅ Statistics dashboard showing total players, average score, and best player
- ✅ Real-time cloud synchronization with Firebase
- ✅ Access data from any device
- ✅ Modern, responsive design

## How to Use

1. Open the application in your web browser
2. Click "Add New Player" to create a player profile
3. Enter player name and average golf score
4. View all players sorted by rank in the table
5. Edit or delete players as needed
6. Access your data from any device!

## Technologies Used

- HTML5
- CSS3 (with modern gradients and animations)
- Vanilla JavaScript
- Firebase Realtime Database

## Setup

Simply open `index.html` in any modern web browser. The app connects to Firebase automatically.

## Firebase Configuration

The app is pre-configured to connect to a Firebase Realtime Database. All player data is stored securely in the cloud.
